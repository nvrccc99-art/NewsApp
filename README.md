# News App

Aplikasi berita Android menggunakan Flutter dan News API.

## Fitur
- Menampilkan berita terbaru dari berbagai sumber
- Detail artikel lengkap
- Refresh untuk update berita terbaru
- Buka artikel di browser

## Setup

1. Install Flutter di sistem Anda
2. Clone/download project ini
3. Jalankan `flutter pub get` untuk install dependencies
4. Jalankan aplikasi dengan `flutter run`
