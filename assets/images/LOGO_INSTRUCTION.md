# Logo Setup Instruction

## Cara menambahkan logo News Now:

1. Simpan gambar logo yang kamu kirim sebagai:
   - `d:\NewsApp\assets\images\logo.png` (untuk resolusi tinggi, minimal 512x512)
   - `d:\NewsApp\assets\images\logo_splash.png` (bisa sama atau versi lebih besar)

2. Setelah file disimpan, hot restart aplikasi (R) untuk melihat hasilnya.

3. Untuk icon aplikasi Android, bisa gunakan tool:
   - flutter_launcher_icons package
   - Atau manual copy ke android/app/src/main/res/mipmap-*/
